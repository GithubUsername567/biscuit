BISCUIT

The pixel pup that runs your Mac.

Hold ⌃ control + fn, say the thing, let go. Biscuit opens the app, plays the song, sets the volume, writes the note — confirms out loud, then goes back to his corner.

▸ TRICKS

Good boy. Very capable boy.

🐾

Pick your pet

Shiba, husky, golden retriever, cat, parrot, seal, or penguin — seven pixel companions, all pettable, all animated. Swap any time in Settings.

🦴

Does, not explains

Ask for a task and Biscuit performs it with real tools — apps, AppleScript, shell — instead of reciting instructions back at you.

🎙️

Hold-to-talk

Hold ⌃+fn anywhere in macOS. Release to send. Stop talking and he figures out you're done. No Enter key, no clicking.

🧠

Local brain

Powered by Ollama running on your own Mac. Your requests never need to leave the machine to get things done.

🗣️

A real voice

Natural speech via Gemini or ElevenLabs TTS, with the system voice as an offline fallback. Replies are one short sentence — by design.

🐕

Stays in his corner

listening working speaking — a 16×16 companion shows state with a glow and a wag. No windows unless you ask.

🔒

Private by default

Speech-to-text runs on-device with Apple's recognizer. No accounts, no telemetry, no cloud required for the core loop.

▸ HOW IT WORKS

Three steps. One of them is just talking.

01

Hold & speak

"Play SZA on YouTube Music." "Set volume to 40." "Make a note: call mom." Biscuit transcribes on-device while you talk.

02

Biscuit fetches

The local model picks tools and runs them — launching apps, opening links, scripting your Mac — chaining steps until the job's done.

03

One bark, then naps

"Done — playing SZA." The bubble fades, the glow goes out, and Biscuit curls back up in the corner until next time.

▸ GET BISCUIT

Bring your own brain (free, local), then build the pup in Xcode.

# 1. the brain (free, runs on your Mac)
$ brew install ollama
$ ollama pull qwen2.5:7b
$ ollama serve

# 2. the dog — clone, open in Xcode, hit run
$ git clone https://github.com/GithubUsername567/biscuit.git
$ cd biscuit
$ open NotchAssistant.xcodeproj

github.com/GithubUsername567/biscuit →

macOS 14+. First run asks for Microphone, Speech Recognition, and Input Monitoring (for the hold-to-talk key). Grant them — he's a good boy.
